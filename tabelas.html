Criando tabelas

Para criação de tabelas é necessário 3 tags. Sendo "table" a primeira ("ultilizar table border ="número" para as bordas da tabela), dentro dela "tr" a segunda e por fim "td". Somente ai podemos escrever os dados da tabela da forma correta
Para cabeçalhos das tabelas ultilizar "th" no lugar de "td" como tag

Cada "tr" é uma linha, então para crição de uma nova linha, usa-se um novo "tr", e posteriormente um novo "td" para o preenchimento
